Use the AI Skill when your coding agent supports `skills` packages.

## Choose your setup path

<Tabs items={["Skills-enabled agent", "MCP-only tool"]}>
  <Tab>
    1. Install the skill:

    ```bash
    npx skills add suiramdev/sdocs
    ```

    2. Restart or reload your agent/tool.
    3. Ask your question normally in your IDE or agent.

  </Tab>
  <Tab>
    If your preferred IDE or assistant only supports MCP server connections, use the [MCP setup guide](./mcp).
  </Tab>
</Tabs>

## What this installs

It adds a reusable `sdocs` skill that gives your agent grounded access to the s&box documentation.
For S&Box C# projects, the skill is applied automatically in compatible tools, so users do not need to mention it explicitly.

## Quick example

Prompt:

```text
How do I spawn an entity in S&Box C#, and what is the exact method signature?
```

The skill helps your agent:

- Search docs quickly
- Explain API entities in plain language
- Return exact signatures when needed
- Answer questions with grounded documentation context
