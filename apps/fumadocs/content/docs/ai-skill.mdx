## What this installs

It adds a reusable `sdocs` skill that gives your agent grounded access to the s&box documentation.
For S&Box C# projects, the skill is applied automatically in compatible tools, so users do not need to mention it explicitly.

## Quick example

Prompt:

```text
How do I spawn an entity in S&Box C#, and what is the exact method signature?
```

The skill helps your agent:

- Search docs first and stay grounded in the index
- Resolve short type names to fully-qualified API symbols
- Inspect type members and exact method overloads before answering
- Return exact signatures, parameter docs, and examples when available
