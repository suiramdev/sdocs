This site is a mirror of the official s&box documentation, rebuilt to provide better API search and tooling for AI agents. It keeps the same API surface and reference content, but adds SDK-style symbol search, member inspection, exact signatures, and machine-friendly responses that make it easier for models to understand and navigate the game code.

## Connect your agent

You can plug your IDE or agent into this documentation through the MCP server.
It exposes a standardized SDK-style toolset for searching the docs, resolving
types, inspecting members, and reading exact method details.

Get started here:

- **Connect via MCP**: [MCP](./mcp)
- **Connect via llms.txt**: [LLM Text](./llms)

Once connected through MCP, your agent can follow a standard SDK workflow:
search the docs, resolve the exact type, inspect members, and then answer using
the exact indexed API metadata.

If your agent does not support MCP, point it at `/llms.txt`. It can use that
index to fetch clean markdown versions of the guide and generated API reference
pages.
